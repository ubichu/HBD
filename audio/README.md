# Audio placeholders

Replace these placeholder files with your own sounds when you are ready:

- `bgm.wav` - looping background music
- `click.wav` - small button click sound
- `envelope-open.wav` - envelope opening sound
- `sparkle.wav` - optional sparkle sound

Keep the same filenames and the JavaScript will work without changes. If you prefer MP3 files, update `AUDIO_FILES` in `js/script.js`.
